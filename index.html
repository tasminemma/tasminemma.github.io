Aims

Through this experiment, I want to achieve a few outcomes:

  1. Create a functional personal website that has:
    • An about me section that lets the right people know the right things.
    • Cool styling. I want the website to be nice to look at (and for the purposes of this project, I don't want to rely on others' Jekyll themes to help me get there).
    • Decent navigation.
  2. Learn HTML, CSS and JavaScript along the way.
    • Earning a certification would be a plus, but not necessary.

Limitations

This website is hosted on Github Pages, which only allows for static websites. It also uses the site builder Jekyll, which I might deactivate with a .nojekyll file (once I figure out if I have to use it, LOL).

Being a static site, I cannot play around with more interactive elements of web development (although at the moment I have literally no idea what that entails), and I worry about losing track of my pages along the way as I keep building this.

I also have my concerns about scaling this up if I were to focus more on the content of my website (e.g. writing, building a portfolio), as coding this manually takes more time than if I were to use a platform like Wordpress to build myself a website.
